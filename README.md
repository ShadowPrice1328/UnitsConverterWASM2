# UnitsConverter
Blazor WebAssembly application for units conversion _[click me](https://shadowprice1328.github.io/UnitsConverterWASM2/)_.

At this moment app offers conversion for:
- Time
- Area
- Length
- Temperature

Other modes will be added soon.
